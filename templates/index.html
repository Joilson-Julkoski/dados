API - DADOS!

Para ultilizar a API, coloque após a barra, a quantidade do dados que voce quer jogar, seguido da letra "d" e então a quantidade de lados que voce que jogar.

 

Exemplo:

https://Minha-API-Entra-21.joilson404.repl.co/5d6
